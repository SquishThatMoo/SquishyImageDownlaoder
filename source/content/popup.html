Settings
Squishy Media Downloader

Press Alt + S to save images / media.

Press Alt + Shift + S to save with options.

These can be changed in the extension settings.

This plugin does not scrape complete galleries, the functionality is limited to single images.

Supported websites
  • DeviantArt
  • e621
  • FurAffinity
  • Hentai Foundry
  • Stuffer DB
  • Twitter
General settings
Must be between 250 - 4000 ms.

Controls the delay between each scraping attempt.

Dark-Mode Auto
  • Off
  • Auto
  • On

Change when dark-mode is active.

Must contain no special chars and end with "/".

Determines the destination folder inside the default download directory.

Additional / create nested folders.

Filename settings
Special characters aren't allowed.

A filename can be generated with following parameters:

$author$
Author
$title$
Title
$number$
Current gallery number
$total$
Total gallery images
$delimiter$
Delimiter - will be removed if the gallery option is disabled.

Special characters /\?%*:|"<>~ will be removed.

Special chars aren't allowed.

The delimiter will be removed if the gallery option is disabled.
